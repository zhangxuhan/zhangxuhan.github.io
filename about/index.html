Software Engineer specializing in embedded systems, desktop applications, and game development.
Passionate about open-source, clean engineering, and building things that actually work — from serial port drivers to private game servers.

What I Do

Full-stack desktop & embedded software development. From low-level driver development (C for Linux serial ports) to high-level application UI (C# WPF/WinForms), and even game script modification for fun.

I focus on solving real problems: getting hardware to communicate, making printers work, optimizing simulation performance — whatever the task demands.

My Approach

Fast design → rapid prototype → iterate and polish. I care about clean architecture and maintainable code, not just "it works."

Comfortable working independently or as part of a team. I document well, communicate clearly, and enjoy tackling technically challenging problems head-on.
